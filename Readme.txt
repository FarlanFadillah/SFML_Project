	- CLICKABLE ANYWHERE TO FIND AN EVIDENCE, GET FINGER PRINT, BLOOD, 	HAIR, ETC
3. OFFICE SCENE
	- PLAYER DESK, SAVE OPTION, ALL CASE FILE, CRIME RECORD (GO TO 	COMPUTER SCENE)
	- FORENSIK ROOM TO ANALYSE DNA RELATED

	- TALK TO SUSPECT

	- ALL DIALOGUE TO NPC OR WITNESSES
	- SOMETIME HAS OPTIONS (DEPENDS ON SCRIPT)

	- SHOWING DISPACHER, WITH ALL INFORMATION RELATED TO THE CASE, 	- 		- CONTINUE BUTTON ( GO TO CRIME SCENE)

	- FIND CRIME RECORD OF PEOPLE
	- MAKE REPORT (OPTIONALLY)
8. LOADING SCENE
	- LOAD CONFIG
	- LOAD ASSETS
